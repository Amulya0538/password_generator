# 🔐 Password Generator

A simple and secure **Password Generator** built using **Python**. This application allows users to generate random passwords based on their preferred length and character types, helping create strong and secure passwords for personal or professional use.

---

## 📌 Features

- Generate passwords of any desired length
- Include uppercase letters (A-Z)
- Include lowercase letters (a-z)
- Include numbers (0-9)
- Include special characters (!@#$%^&*)
- User-friendly command-line interface
- Input validation for password length
- Random and secure password generation

---

## 🛠️ Technologies Used

- Python 3
- `random` module
- `string` module
